# Analysis Skill Module

This module serves as the central hub for the Senior Data Scientist's analytical capabilities. Use this entry point to locate specific methods.

## Available Capabilities

### 1. Statistics & Hypothesis Testing
**File**: `skills/analysis/statistics.md`
-   **Use For**: A/B testing, regression, causal inference, determining statistical significance.
-   **Key Methods**: T-Tests, ANOVA, Linear/Logistic Regression.

### 2. Exploratory Data Analysis (EDA)
**File**: `skills/analysis/eda.md`
-   **Use For**: Initial data investigation, quality checks, distribution analysis.
-   **Key Methods**: `pandas_profiling`, correlation matrices, missing value analysis.

### 3. Explainable AI (XAI)
**File**: `skills/analysis/xai.md`
-   **Use For**: Interpreting model predictions, feature importance, debugging complex models.
-   **Key Methods**: SHAP (Tree/Deep/Kernel/Linear), Waterfall plots, Beeswarm plots.

### 4. KPI Design
**File**: `skills/analysis/kpi.md`
-   **Use For**: Defining success metrics, structuring dashboards, selecting SMART KPIs.
-   **Key Methods**: Executive vs. Operational metrics, Department-specific KPIs.

## How to Use
1.  **Identify the need**: e.g., "Why did this user churn?"
2.  **Select the sub-skill**: e.g., "I need `xai.md` to see top features."
3.  **Execute**: Load the sub-skill and follow its specific quick start guide.
